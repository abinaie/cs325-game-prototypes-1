The Little Gold Miner

  • Originality: It is basically a Snakes and Ladders Game. It is a single player game. The player wins if they be able to get to the trophy.
  • Technical Merit: It is challenging for the player to get to the last block in a short amount of time because of the ladders that leads to lower blocks.
  • Prototype Postmortem: I learned a little more of JavaScript as well as some Phaser game Logics and functions. This is my best game so far and I think it is partly because we had more time for it. This time I tried to design this one based on my physical prototype that I created last week instead of modifying an existing one.
  • Prototype Assets:
    • Tiled Tutorial: source code
    • Music: audio
    • Sprite: robot art
    • Trophy: item art
    • Dice: item art
  • Prototype Closest Other Game: It is similar to the Snakes and Ladders game.
  • High Concept: The game's goal is to play a little dice/board game for fun! I will try to make it two players game for the next iteration.
  • Theme: Board Game!
  • Mandated Variety: Input : Keyboard -- Randomness: Dice! -- Genre : Board Game. The play style I used in this prototype is the Collector style.
  • Prototype Goal: This prototype is used to improve the game developing skills of the developer.
  • Player Experience Goals: So the player has some fun playing a board game!
  • Game play: This is a single player game. The player starts the game by throwing the dice and move the game character accordingly. If the roll of dice leads to a red ladder it takes the player back some number of blocks and if the the roll of dice leads to a green ladder it takes the player some blocks closer to the trophy.
  • Strategies: All you need is a bit of luck!
  • Story/Setting/Premise: The story of a young man who borrowed money, mortgaged his property or spent his life savings to make the arduous journey to California. In pursuit of the kind of wealth he had never dreamed of, he left his families and hometowns to find gold. The player will experience hardships and rewards of this journey. The player could find some gold if they got lucky and get there faster than the others. The player toss a dice and moves the player accordingly through this journey.
  • Target Audience: Anyone that could work with the keyboard!
  • Play Time: It should take around 2-3 minutes!
