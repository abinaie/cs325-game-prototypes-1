The Garbage Collector!

  • Originality: It is basically a Snake Game. It is a single player game. The player wins if they be able to collect all of the garbages in the ocean.
  • Technical Merit: It is challenging for the player to control the motion of the garbage collector, since left key goes right and down key goes up and etc. However the speed of collector is reduced.
  • Prototype Postmortem: I learned a little more of JavaScript as well as some Phaser game Logics and functions.
  • Prototype Assets: This game uses the assets of the Snake Game and some added challenges and colors.
    • Snake Game: here!
  • Prototype Closest Other Game: It is similar to the snake game example plus some extra added features.
  • High Concept: To understand how hard it is to track garbages dump in the middle of the ocean!
  • Theme: Snake Game!
  • Mandated Variety: Input : Arrow Keys -- Randomness: The location the ocean waves will take garbages is random! -- Genre : Cleaning. The play style I used in this prototype is the Collector style.
  • Prototype Goal: This prototype is used to improve the game developing skills of the developer.
  • Player Experience Goals: So the player has some fun playing and also to increase their environmental awareness and to understand the fragility of our environment!
  • Game play: This is a single player game. The player should try to control the garbage collector ship among heavy waves of the pacific ocean and collect all the garbages left behind. However it is not easy to control it you would like go up it goes down and you would like to go left it goes right and so forth.
  • Strategies: The best strategy is to practice to get used to the controller.
  • Story/Setting/Premise: The moving part is the garbage collector. The garbage collector gets heavier and longer as it collects more trashes. The player could control the game using the 4 arrow keys.
  • Target Audience: Anyone could play this game!
  • Play Time: It depends! It could be challenging to get used to the controller.
